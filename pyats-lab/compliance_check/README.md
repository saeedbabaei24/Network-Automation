# VLAN Compliance Check (pyATS)

This project connects to Cisco switches using pyATS,
collects VLAN information, and compares each switch
against a site baseline switch.

## Features
- VLAN database comparison
- Per-site baseline
- Missing / extra VLAN detection
- VLAN name mismatch detection

## Requirements
- Python 3.9+
- pyATS
- Genie

## Usage

1. Create virtual environment
2. Install dependencies
3. Update devices.yaml
4. Run the script
